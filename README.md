# Det_Kaerlige_Maaltid

Github for projektet Det Kærlige Måltid.

Her er nogle guidelines til kodningen.

RESPONSIVT DESKTOP DESIGN
Vi laver et responsivt design der passer til udelukkende Desktop.
Vi tager udgangspunkt i den MINDSTE/STØRSTE størrelse desktop i vores kodning, og andre størrelser justerer
vi i media queries.

Mindste størrelse desktop vi laver design til:
1366 x 768

Største størrelse desktop vi laver design til:
1920 x 1080

---

FONTE
Til overskrifter:
font-family:'Source Serif Pro', serif;

Til brødtekst:
font-family: 'Montserrat', sans-serif;

---

NAVNGIVNING AF CLASSES OG ID
Vær specifik med navngivningen af classes og ID. Undgå navne som "tekst" "billede" "illustration" osv.
Prøve istedet "Header_brødtekst" "billede_pakning_af_mad" "illustration_familie"
